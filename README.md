## Installation

```bash
$ npm install
```

## Running the Docker with PostgreSQL service

**Install docker and docker-compose using the instruction https://docs.docker.com/get-docker/**

```bash
# for start
$ docker-compose up

# for stop
$ docker-compose down
```

## Running the backend service

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Seed DB

```bash
$ npm run seed
```
